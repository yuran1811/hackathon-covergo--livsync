# LivSync - AI Health Assistant

An intelligent health assistant that combines real-time calendar monitoring with personalized health insights using AI.

## Key Features

### **AI-Powered Health Insights**
- **Google Gemini 2.5 Flash** integration for intelligent health analysis
- **Multi-Agent AI System** with specialized health and calendar agents
- **Personalized Recommendations** based on user health data and objectives

### **Smart Calendar Integration**
- **Real-time Event Monitoring** with intelligent polling
- **AI Event Suggestions** that align with health goals
- **Automatic Schedule Optimization** when events change

### **Comprehensive Health Tracking**
- **Health Metrics Correlation** - steps, sleep, stress, heart rate, blood pressure
- **Realistic Health Data Simulation** with medical correlations
- **Weekly Workout History** and activity tracking

### **Event-Driven Architecture**
- **Proactive AI Responses** to calendar changes
- **Async Processing** for real-time health suggestions
- **Intelligent Backoff** strategies for API optimization

## Tech Stack

- **Frontend**: React + Vite + TypeScript + Tailwind CSS
- **Backend**: FastAPI + Python
- **AI**: LangChain + Google Gemini 2.5 Flash
- **Database**: Supabase
- **Calendar**: Nylas API

## Development

# Install dependencies

# Start api server

# Start web app
